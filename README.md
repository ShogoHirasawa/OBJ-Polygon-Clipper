# OBJ Polygon Clipper

Minecraftのワールドデータを3Dモデル（OBJファイル）に変換した後、**好きな形の範囲だけを切り出せる**デスクトップアプリです。

Mineways（Minecraft→OBJ変換ツール）では四角形の範囲しか選べませんが、このツールを使えば道路に沿った形や自由な多角形で切り出すことができます。

---

## 前提条件

以下のソフトウェアがパソコンにインストールされている必要があります。

### 1. Node.js（v18以上）

Node.js はこのアプリを動かすために必要です。

1. https://nodejs.org/ にアクセス
2. 「LTS」と書かれた緑色のボタンをクリックしてダウンロード
3. ダウンロードしたファイルを開いてインストール（すべて「次へ」でOK）

インストールできたか確認するには、ターミナル（Mac）またはコマンドプロンプト（Windows）を開いて以下を入力してください：

```
node -v
```

`v18.0.0` のようにバージョンが表示されればOKです。

### 2. Git

GitHubからファイルをダウンロードするために必要です。

- **Mac**: ターミナルで `git --version` と入力してください。初回は自動でインストールされます。
- **Windows**: https://git-scm.com/ からダウンロード・インストールしてください。

### 3. Mineways

Minecraftのワールドを OBJ ファイルに変換するためのツールです。

- https://mineways.com/ からダウンロード

## セットアップ手順

### 手順1: このリポジトリをダウンロード

ターミナル（Mac）またはコマンドプロンプト（Windows）を開いて、以下のコマンドを1行ずつ入力してください。

```bash
git clone https://github.com/ShogoHirasawa/mincraftTo3dtiles.git
```

```bash
cd mincraftTo3dtiles
```

### 手順2: 必要なパッケージをインストール

```bash
npm install
```

初回のみ数分かかります。完了するまで待ってください。

### 手順3: アプリを起動

```bash
npm start
```

アプリのウィンドウが表示されます。

## 使い方

### ステップ1: OBJファイルを用意する

1. Minewaysを起動して Minecraft のワールドを開く
2. 変換したい範囲を四角形で選択（大きめでOK）
3. File → Export for Rendering で OBJ 形式でエクスポート
4. `.obj` ファイル、`.mtl` ファイル、`tex` フォルダが作成されます

### ステップ2: OBJファイルを読み込む

1. アプリの「OBJを開く」ボタンをクリック
2. Minewaysで作成した `.obj` ファイルを選択
3. プログレスバーが表示され、読み込みが完了するとモデルが3Dで表示されます

### ステップ3: 視点を調整する

- **ドラッグ（左ボタン）**: 回転
- **右ドラッグ**: パン（移動）
- **スクロール**: ズーム

切り出したい範囲が見やすい角度に調整してください。

### ステップ4: 切り出す範囲を選択する

1. 「選択範囲を描く」ボタンをクリック
2. モデルの上で**左クリック**して、切り出したい範囲の頂点を追加していきます
3. 最低3点をクリックしたら、**右クリック**または「確定」ボタンで範囲を確定します

確定すると、赤い半透明の立体が表示されます。これが実際に切り出される範囲です。

### ステップ5: 高さを調整する（任意）

右上のスライダーで、切り出す高さを調整できます。
デフォルトではモデルの一番高い所まで含まれます。

### ステップ6: エクスポート

1. 「エクスポート」ボタンをクリック
2. 保存先を選択
3. 選択範囲内のモデルが `.obj` ファイルとして保存されます
4. テクスチャ（`.mtl` ファイルと `tex` フォルダ）も自動的にコピーされます

---

## 操作まとめ

| 操作 | 説明 |
|------|------|
| 左ドラッグ | カメラ回転 |
| 右ドラッグ | カメラ移動 |
| スクロール | ズーム |
| 左クリック（描画中） | 選択範囲の頂点を追加 |
| 右クリック（描画中） | 選択範囲を確定 |
| 「戻す」ボタン | 最後の頂点を取り消し / 確定を取り消し |
| 「クリア」ボタン | 選択範囲をすべて消去 |

## うまくいかないとき

### アプリが起動しない
- `npm install` を再度実行してみてください
- Node.js のバージョンが古い場合は最新版に更新してください

### モデルが表示されない
- OBJファイルのサイズが非常に大きい場合は読み込みに時間がかかります（プログレスバーを確認）
- Minewaysの設定で「Center model: YES」にしてエクスポートしてください

### テクスチャが表示されない
- `.obj` ファイルと同じフォルダに `.mtl` ファイルと `tex` フォルダがあることを確認してください
